# Normal Card Replacement Icon Candidates

This folder contains candidate replacement icons extracted from the provided `Normal.png` card image.

These files do not replace the active Canon icon set. They are staged here for visual review and possible future adoption.

- `Originals/`: untouched crops from the source image
- `PNG/`: normalized 64 x 64 transparent PNG icons with black artwork
- `SVG/`: vector-only SVG approximations generated from the normalized masks
- `icon-map-candidate.yaml`: candidate field-to-icon mapping
- `manifest.yaml`: extraction metadata and confidence notes
- `preview.html`: visual review page
