# Photography Icon Assets

This folder contains icon infrastructure for the Photography Reference System.

Icons are stored in `Photography Icons/` and referenced by `icon-map.yaml`.

The SVG files are from Feather Icons, an MIT-licensed open-source icon set:
https://feathericons.com
