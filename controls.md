- Lens Control Ring assignment page, with the row selected and any assignment detail/options screen.
- Movie Record button assignment page in each available shooting context.
- `Touch & Drag AF` page and every subordinate settings page.
- `Multi-function lock` page showing every lock checkbox/state.
- Playback custom-control/custom-function pages, with every programmable playback control selected.
- Physical photographs of the attached lens showing the Lens AF/MF switch and Lens IS switch positions; repeat for each lens whose permanent switch configuration is to be documented.
- Every My Menu tab/page, scrolled as needed to show every registered item and page name.
- Custom shooting mode registration/settings screens for C1, C2, and C3, plus each mode recalled on the shooting screen and any displayed registered-mode description.
- Customized Quick Control Screen configuration page(s) and the resulting Quick Control Screen.
- Custom display configuration pages, including shooting-info display customization and every enabled display layout.
- Any remaining programmable-control page not included above, including controls added or changed by the installed firmware or attached accessories.
