Image Stabilization

Purpose

Explain how image stabilization works on the Canon EOS R5 and when to use Mode 1, Mode 2, Mode 3, or Off.

On the Canon EOS R5, image stabilization can come from three places:

  • Lens IS in supported lenses.
  • IBIS (In-Body Image Stabilization) in the camera.
  • Coordinated IS when a supported RF lens and the camera body work together.

What it Does

Image stabilization reduces blur from camera movement. It does not freeze subject motion; shutter speed still controls motion blur from the subject.

The camera offers several stabilization behaviors depending on the lens and shooting mode.

ModeWhat it stabilizesBest use
Mode 1 (Continuous IS)Vertical + horizontal movementGeneral photography, portraits, landscapes, travel, handheld macro, low-light handheld work
Mode 2 (Panning IS)Vertical movement only during horizontal pansMotorsports, airshows, birds flying steadily across the frame, cyclists, running subjects while panning
Mode 3 (Exposure-only IS)Stabilization activates fully during exposureErratic action where a stabilized viewfinder can make tracking harder, including birds, wildlife, and sports
OffNo stabilizationSolid tripod work, multi-minute exposures, astrophotography, very high shutter speeds, and certain long exposures

How it Works

Lens IS moves lens elements to counter camera shake. IBIS moves the camera sensor. With compatible RF lenses, the R5 coordinates lens IS and IBIS automatically, so the main field decision is usually the IS mode or whether stabilization should be off.

Mode 1 stabilizes continuously. Mode 2 ignores intentional panning movement in one direction. Mode 3 keeps the view less stabilized while tracking and applies stabilization during the exposure.

Advantages

  • Mode 1 is the best default for most handheld photography.
  • Mode 2 supports intentional motion blur while panning.
  • Mode 3 can make erratic subjects easier to track.
  • Turning IS off avoids stabilization drift during some tripod and long-exposure work.
  • Coordinated IS can improve handheld results with supported RF lenses.

Disadvantages

  • Stabilization cannot stop subject motion.
  • Mode 1 can create a floating viewfinder feel with erratic subjects.
  • Mode 2 is specialized and should not be used as a general default.
  • Stabilization can be unnecessary or counterproductive on a solid tripod.
  • Some lenses do not provide all modes.

Recommended Uses

Use Mode 1 for landscapes, portraits, street photography, travel, handheld macro, and low-light handheld work. This is the default mode and should be used most of the time.

Use Mode 2 for intentional panning, such as a race car at 1/60 sec where the background blurs while the car stays sharp.

Use Mode 3 for birds in flight, football, soccer, basketball, and wildlife moving unpredictably. Many wildlife photographers leave long telephotos in Mode 3.

Turn IS off for solid tripod work, multi-minute exposures, astrophotography, and situations where shutter speeds are already very fast, such as 1/2000-1/4000 sec.

When Not to Use

Do not leave stabilization on automatically for every tripod setup. Modern Canon RF lenses generally detect tripod use well, but for completely static tripod shots, turning stabilization off can still be beneficial.

Do not rely on IS to solve subject blur. If the subject is moving, raise shutter speed or change timing.

Avoid Mode 2 unless you are intentionally panning.

Decision Guide

If handheld and the subject is mostly static, use Mode 1.

If intentionally panning, use Mode 2.

If tracking fast erratic subjects, use Mode 3.

If on a solid tripod for static or long-exposure work, turn IS off.

If using an RF lens with IS, let the R5 coordinate lens IS and IBIS; choose the mode based on the subject.

Canon-Specific Notes

With RF lenses that include IS, the R5 automatically coordinates lens and body stabilization. You do not normally choose lens IS versus IBIS separately in the field.

Some lenses expose a physical IS switch and mode selector. Lens controls can determine whether the camera menu offers the expected options.

EF 100-400mm f/4.5-5.6L IS II USM

For the Canon EF 100-400mm f/4.5-5.6L IS II USM, commonly adapted to the R5 with a Canon EF-EOS R adapter, stabilization is controlled by the switch on the lens.

When this lens is used on the R5 with the Canon adapter, the lens IS and the camera's IBIS work together. In the field, treat the lens IS mode switch as the primary stabilization mode choice.

Lens IS ModePurposeBest use
Mode 1Stabilizes in all directionsGeneral wildlife, perched birds, landscapes, travel, handheld telephoto
Mode 2Panning modeBirds flying smoothly across the frame, motorsports, cyclists, runners
Mode 3Stabilization only during exposureBirds in flight, erratic wildlife, field sports, fast action
OffNo stabilizationRigid tripod work, very long exposures, or very fast shutter speeds where IS adds little practical benefit

Use Mode 1 for static or slow-moving subjects: perched birds, wildlife at rest, zoo photography, travel, handheld landscapes, and general telephoto work.

Use Mode 2 for intentional panning. The lens detects the panning direction and stabilizes only perpendicular to that movement. This is the right choice for horizontal pans of birds in flight, airshows, motorsports, cyclists, and running animals.

Use Mode 3 for action that changes direction quickly. The viewfinder remains largely unstabilized while tracking, and full stabilization is applied only when the shutter is released. This is often preferred for birds in flight, erratic wildlife, sports, and subjects that are difficult to keep in the frame.

Many wildlife photographers leave this lens in Mode 3 for unpredictable subjects because it makes tracking feel more natural.

For completely static tripod work, long exposures, and astrophotography, turning stabilization off remains a good practice.

For this reference system, treat Mode 1 as the default, Mode 2 as the dedicated panning mode, and Mode 3 as the default for fast action.

Tips

  • Use Mode 1 as the everyday handheld default.
  • Use Mode 3 for long telephoto wildlife and unpredictable action.
  • Use Mode 2 only when you are deliberately panning.
  • Recheck IS after switching between handheld and tripod work.
  • Treat stabilization as camera-shake reduction, not a substitute for shutter speed.

Common Mistakes

  • Leaving Mode 1 on for erratic action when Mode 3 would track better.
  • Using Mode 2 for non-panning subjects.
  • Leaving IS on during long static tripod exposures.
  • Expecting stabilization to freeze subject movement.
  • Forgetting that some lenses do not provide Mode 2 or Mode 3.
