Official Canon names and extracted assets from Canon EOS R5 Product Manual pages at cam.start.canon.

Shooting Modes

Scene Intelligent Auto
A+
Fully automatic shooting mode where the camera analyzes the scene and sets optimum settings automatically.
Source: Canon EOS R5 manual
Mode selection
MODE
Generic shooting mode selector used to choose Canon EOS R5 exposure modes.
Source: Canon EOS R5 manual
Flexible-priority AE
Fv
User can set shutter speed, aperture value, ISO speed, and exposure compensation manually or automatically.
Source: Canon EOS R5 manual
Program AE
P
Camera automatically sets shutter speed and aperture value to suit subject brightness.
Source: Canon EOS R5 manual
Shutter-priority AE
Tv
User sets shutter speed while the camera sets aperture for standard exposure.
Source: Canon EOS R5 manual
Aperture-priority AE
Av
User sets aperture while the camera sets shutter speed for standard exposure.
Source: Canon EOS R5 manual
Manual exposure
M
User sets both shutter speed and aperture value and checks exposure with the exposure level indicator.
Source: Canon EOS R5 manual
Bulb
BULB
Shutter stays open while the shutter button is held down, for long exposures such as night scenes or fireworks.
Source: Canon EOS R5 manual
Custom shooting mode 1
C1
Registered camera settings can be recalled as custom shooting mode C1.
Source: Canon EOS R5 manual
Custom shooting mode 2
C2
Registered camera settings can be recalled as custom shooting mode C2.
Source: Canon EOS R5 manual
Custom shooting mode 3
C3
Registered camera settings can be recalled as custom shooting mode C3.
Source: Canon EOS R5 manual
