Fv (Flexible Priority)

What is Fv?

Fv (Flexible Priority) is Canon's Flexible Priority shooting mode. Unlike traditional exposure modes, Fv allows each exposure parameter to be independently set to either Manual or AUTO.

This lets you move between fully automatic exposure, selective control, and manual exposure without switching among P, Av, Tv, and M.

Every parameter may be controlled individually. Setting one parameter manually does not require the others to be manual.

How to Change Settings

  1. Press INFO until the exposure parameter line is active.
  2. Select Shutter, Aperture, ISO, or Exposure Compensation.
  3. Rotate the Main Dial to change the selected value.
  4. Press SET to return that parameter to AUTO.

Why Use Fv?

Fv allows photographers to stay in a single shooting mode while deciding which exposure parameters are automatic and which are manually controlled. It effectively replaces frequent switching among P, Av, Tv, and much of M mode while preserving direct control whenever it is needed.
