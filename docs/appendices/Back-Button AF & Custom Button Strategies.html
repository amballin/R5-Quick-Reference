Custom Controls, Back-Button AF & Dial Strategies

Purpose

Document a consistent Canon EOS R5 button, dial, back-button autofocus, and exposure-control layout for birds, wildlife, sports, macro, landscape, travel, and general field use. This is a recommended configuration, not a record of settings verified from the current-camera screenshots.

What it Does

Back-button AF separates autofocus activation from the shutter release. The recommended layout uses two independent AF-start buttons: AF-ON starts autofocus with the active profile settings, while the AE Lock button starts autofocus with a detailed 1-Point AF override. Choose one AF button at a time, then use the shutter to take the picture.

Custom controls can also assign frequently used functions to M-Fn, the joystick, the lens or adapter Control Ring, and Register/Recall Shooting Function. Keep one consistent base layout and add a specialized assignment only when it solves a repeated field problem.

How it Works

With Shutter button half-press assigned to Metering start, pressing the shutter button does not refocus the camera. Press or hold either configured rear AF button, then press the shutter to take the picture.

Set AF-ON to Metering and AF start with its INFO details left at Maintain current setting. It then follows the active profile: for example, Face + Tracking with Animals and Eye Detection in a wildlife profile.

Set the AE Lock button to Metering and AF start and use INFO details to keep the current AF Operation and Servo characteristics while overriding only AF Method to 1-Point AF. This button starts the alternate autofocus itself. Do not hold AF-ON and AE Lock together.

Switch to registered AF function can temporarily change selected AF behavior, but it is not the recommended default here because it may require the override button, a separate AF-start button, and the shutter at the same time. Register/Recall Shooting Function is broader, but its original-R5 item list and AF-start behavior remain under evaluation and should not be recommended until verified directly on the camera.

The SET button provides a consistent recovery action when the active AF point has moved away from the desired starting position. The Depth-of-field preview button provides direct access to the selected One-Shot AF/Servo AF change.

Advantages

  • Separates focus activation from image capture.
  • Stops autofocus immediately when the selected rear AF button is released.
  • Provides normal tracking and precise 1-Point AF from separate AF-start buttons without navigating menus.
  • Requires only one rear AF button at a time.
  • Provides a one-press method to return the AF point to center.
  • Keeps direct AF-point movement on the joystick and frequently changed secondary controls within reach of M-Fn.
  • Keeps shutter speed, aperture, ISO speed, and exposure compensation on separate direct controls.
  • Makes exposure compensation directly available when using Manual exposure with Auto ISO.

Disadvantages

  • Requires deliberate use of AF-ON; half-pressing the shutter no longer starts autofocus.
  • Both AF-start buttons must be configured through their INFO detail screens and remembered.
  • The AE Lock button is no longer available for its default purpose while reassigned.
  • The Depth-of-field preview button is no longer available for its default purpose while reassigned.
  • A customized camera can be confusing to another photographer unless the layout is documented.

Recommended Uses

  • Birds in flight and wildlife when Face + Tracking is normal but deliberate 1-Point AF is sometimes needed.
  • Sports when the subject must be tracked through changing backgrounds.
  • Perched birds or partially obscured subjects when a temporary alternate AF behavior is useful.
  • Manual exposure with Auto ISO when direct exposure compensation is required.
  • Any profile that benefits from one consistent physical-control layout.

When Not to Use

  • Do not enable the layout immediately before a critical shoot without practicing it.
  • Do not use back-button AF if shutter-button autofocus is preferred or required by another camera user.
  • Do not use Switch to registered AF function as the default alternate-AF control unless its multi-button ergonomics have been tested with the intended lens.
  • Do not recommend Register/Recall Shooting Function until its original-R5 behavior and participating settings have been documented.
  • Do not rely on exposure compensation in Manual exposure unless Auto ISO is active.
  • Do not assign a broad Register/Recall state until every recalled setting has been documented and tested.

These recommendations define control behavior only. AF method, subject detection, eye detection, and other shooting settings remain owned by the baseline and subject-profile overrides.

Canon-Specific Notes

  • Use the Canon EOS R5 Customize buttons and Customize dials menus for these assignments.
  • Canon menu wording can vary slightly by firmware display context. Confirm the function shown on the camera before relying on it.
  • Metering and AF start can be configured through the INFO detail screen on supported controls. For AF-ON, maintain the active profile settings; for AE Lock, override only AF Method to 1-Point AF.
  • Eye Detection on the original EOS R5 requires Face + Tracking. It is not active while the 1-Point AF override is in use.
  • Subject to Detect takes effect with Face + Tracking, Zone AF, and Large Zone AF; it does not need to be changed when the alternate button selects 1-Point AF.
  • Switch to registered AF function and Register/Recall Shooting Function remain advanced alternatives rather than parts of this recommended layout.
  • The Control Ring may be on an RF lens or an EF-EOS R control ring mount adapter.
  • This recommended layout is separate from the evidence-based current configuration documented in Canon EOS R5 Custom Controls Current Configuration.

Tips

  • Practice pressing AF-ON before removing autofocus from the shutter button.
  • Keep the same base button and dial layout across subject profiles.
  • Configure and test AF-ON first, then configure AE Lock with only the AF Method override.
  • Practice choosing one rear AF button without pressing the other.
  • Test exposure compensation with Manual exposure and Auto ISO before field use.
  • Recheck assignments after a camera reset or firmware-related settings reset.

Common Mistakes

  • Removing autofocus from the shutter button before assigning and testing AF-ON.
  • Treating a recommended assignment as confirmation of the camera's current state.
  • Pressing AF-ON and AE Lock together instead of choosing one AF behavior.
  • Configuring the AE Lock detail screen to override more settings than intended.
  • Expecting Eye Detection to operate while the 1-Point AF override is active.
  • Expecting exposure compensation to change a fully manual exposure with fixed ISO.
  • Duplicating the full control explanation inside every subject profile.
