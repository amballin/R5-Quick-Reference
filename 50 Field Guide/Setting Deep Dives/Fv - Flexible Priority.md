# Fv (Flexible Priority)

## What is Fv?

Fv (Flexible Priority) is Canon's Flexible Priority shooting mode. Unlike traditional exposure modes, Fv allows each exposure parameter to be independently set to either **Manual** or **AUTO**.

This lets you move between fully automatic exposure, selective control, and manual exposure without switching among P, Av, Tv, and M.

## Parameters

| Setting | AUTO | Manual |
|---|:---:|:---:|
| Shutter Speed | ✓ | ✓ |
| Aperture | ✓ | ✓ |
| ISO | ✓ | ✓ |
| Exposure Compensation | ✓ | ✓ |

Every parameter may be controlled individually. Setting one parameter manually does not require the others to be manual.

## How to Change Settings

1. Press ![INFO](../../60 Assets/icons/canon_r5_official/info.svg) **INFO** until the exposure parameter line is active.
2. Select **Shutter**, **Aperture**, **ISO**, or **Exposure Compensation**.
3. Rotate the **Main Dial** to change the selected value.
4. Press **SET** to return that parameter to **AUTO**.

## Display Example

| Parameter | Displayed Value |
|---|---:|
| Tv | 1/1000 |
| Av | F5.6 |
| ISO | 400 |
| EC | +0.3 |

> **Note:** Any parameter displaying **AUTO** is currently being controlled by the camera.

## Common Configurations

| Configuration | Tv | Av | ISO | Result |
|---|---:|---:|---:|---|
| P Mode Equivalent | AUTO | AUTO | AUTO | Camera controls everything. |
| Sports | 1/2000 | AUTO | AUTO | Locks shutter speed. The camera adjusts aperture and ISO. |
| Portrait | AUTO | F2.8 | AUTO | Locks aperture. The camera adjusts shutter speed and ISO. |
| Landscape | AUTO | F11 | 100 | Locks aperture and ISO. The camera adjusts shutter speed. |
| Manual + Auto ISO | 1/1000 | F5.6 | AUTO | The photographer controls motion blur and depth of field while Auto ISO maintains exposure. Useful for wildlife, birds, sports, and action. |

## Fv Compared to Traditional Modes

| Traditional Mode | Equivalent in Fv |
|---|---|
| P | Tv=AUTO, Av=AUTO, ISO=AUTO |
| Av | Tv=AUTO, Av=Manual, ISO=AUTO |
| Tv | Tv=Manual, Av=AUTO, ISO=AUTO |
| M + Auto ISO | Tv=Manual, Av=Manual, ISO=AUTO |
| Manual | Tv=Manual, Av=Manual, ISO=Manual |

## Why Use Fv?

Fv allows photographers to stay in a single shooting mode while deciding which exposure parameters are automatic and which are manually controlled. It effectively replaces frequent switching among P, Av, Tv, and much of M mode while preserving direct control whenever it is needed.
