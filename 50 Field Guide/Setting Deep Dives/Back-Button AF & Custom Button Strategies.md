# Back-Button AF & Custom Button Strategies

## Purpose

Document a consistent Canon EOS R5 back-button autofocus and exposure-control layout for birds, wildlife, sports, macro, and general field use. This is a recommended configuration, not a record of settings verified from the current-camera screenshots.

## What it Does

Back-button AF separates autofocus activation from the shutter release. The recommended layout keeps normal autofocus on **AF-ON**, places a temporary alternate AF behavior on the **AE Lock button**, and assigns recovery and AF-operation changes to controls that can be reached without opening a menu.

### Recommended Custom Button Layout

| Control | Canon assignment | Purpose |
|---|---|---|
| **Shutter button half-press** | **Metering start** | Starts metering without starting autofocus. |
| **AF-ON button** | **Metering and AF start** | Starts the normal autofocus configuration. |
| **AE Lock button** | **Switch to registered AF function** | Recalls the configured registered AF function while the button is held. |
| **SET button** | **Set AF point to center** | Returns the active AF point to the center. |
| **Depth-of-field preview button** | **One-Shot AF ↔ Servo AF** | Changes AF operation between One-Shot AF and Servo AF. |

### Recommended Dial Layout

| Control | Assigned function |
|---|---|
| **Main Dial (around Shutter Button)** | **Shutter Speed (Tv)** |
| **Quick Control Dial (Rear Wheel)** | **Aperture (Av)** |
| **Multi-function Dial (Top Rear Dial)** | **ISO Speed** |
| **Control Ring (Lens or Control Ring Adapter)** | **Exposure Compensation** |

## How it Works

With **Shutter button half-press** assigned to **Metering start**, pressing the shutter button does not refocus the camera. Press or hold **AF-ON** when autofocus is required, then release it when autofocus should stop.

**Switch to registered AF function** provides a temporary alternate AF behavior configured in the camera. Hold the assigned **AE Lock button** when the alternate behavior is needed; release the button to return to the normal AF behavior. The registered choices must be configured and tested on the camera before field use.

The **SET button** provides a consistent recovery action when the active AF point has moved away from the desired starting position. The **Depth-of-field preview button** provides direct access to the selected One-Shot AF/Servo AF change.

## Advantages

- Separates focus activation from image capture.
- Stops autofocus immediately when **AF-ON** is released.
- Provides a temporary alternate AF behavior without navigating menus.
- Provides a one-press method to return the AF point to center.
- Keeps shutter speed, aperture, ISO speed, and exposure compensation on separate direct controls.
- Makes exposure compensation directly available when using Manual exposure with Auto ISO.

## Disadvantages

- Requires deliberate use of **AF-ON**; half-pressing the shutter no longer starts autofocus.
- The registered AF function must be configured and remembered.
- The **AE Lock button** is no longer available for its default purpose while reassigned.
- The **Depth-of-field preview button** is no longer available for its default purpose while reassigned.
- A customized camera can be confusing to another photographer unless the layout is documented.

## Recommended Uses

- Birds in flight and wildlife when continuous tracking is the normal behavior.
- Sports when the subject must be tracked through changing backgrounds.
- Perched birds or partially obscured subjects when a temporary alternate AF behavior is useful.
- Manual exposure with Auto ISO when direct exposure compensation is required.
- Any profile that benefits from one consistent physical-control layout.

## When Not to Use

- Do not enable the layout immediately before a critical shoot without practicing it.
- Do not use back-button AF if shutter-button autofocus is preferred or required by another camera user.
- Do not assign **Switch to registered AF function** until the registered behavior has been configured and tested.
- Do not rely on exposure compensation in Manual exposure unless Auto ISO is active.

## Decision Guide

| Situation | Control | Action |
|---|---|---|
| Start normal autofocus or continuous tracking | **AF-ON button** | Press or hold **Metering and AF start**. |
| Temporarily use the registered AF behavior | **AE Lock button** | Hold **Switch to registered AF function**; release to return. |
| Return the active AF point to center | **SET button** | Press **Set AF point to center**. |
| Change between stationary and continuous AF operation | **Depth-of-field preview button** | Use **One-Shot AF ↔ Servo AF**. |
| Change shutter speed | **Main Dial (around Shutter Button)** | Rotate the dial. |
| Change aperture | **Quick Control Dial (Rear Wheel)** | Rotate the dial. |
| Change ISO speed | **Multi-function Dial (Top Rear Dial)** | Rotate the dial. |
| Adjust exposure with Manual + Auto ISO | **Control Ring** | Apply **Exposure Compensation**. |

## Recommended Settings by Profile

| Profile | Normal use | Temporary or recovery use |
|---|---|---|
| **Birds in Flight** | Hold **AF-ON** for continuous tracking. | Hold **AE Lock** for the configured registered AF behavior; press **SET** to recenter the AF point. |
| **Birds Perched** | Use **AF-ON** for deliberate focus activation. | Use **One-Shot AF ↔ Servo AF** when AF operation needs to change. |
| **Wildlife** | Hold **AF-ON** while the subject is moving. | Hold **AE Lock** when the registered AF behavior is better suited to an obstructed or difficult scene. |
| **Sports** | Hold **AF-ON** for continuous tracking. | Use the registered AF behavior for a temporary alternate AF response. |
| **Macro** | Use **AF-ON** only when autofocus is wanted. | Use **One-Shot AF ↔ Servo AF** only when the subject or camera movement requires the change. |

These recommendations define control behavior only. AF method, subject detection, eye detection, and other shooting settings remain owned by the baseline and subject-profile overrides.

## Canon-Specific Notes

- Use the Canon EOS R5 **Customize buttons** and **Customize dials** menus for these assignments.
- Canon menu wording can vary slightly by firmware display context. Confirm the function shown on the camera before relying on it.
- **Switch to registered AF function** recalls only the options selected in its camera configuration. This appendix does not prescribe unverified registered values.
- The Control Ring may be on an RF lens or an EF-EOS R control ring mount adapter.
- This recommended layout is separate from the evidence-based current configuration documented in **Canon EOS R5 Custom Controls Current Configuration**.

## Tips

- Practice pressing **AF-ON** before removing autofocus from the shutter button.
- Keep the same base button and dial layout across subject profiles.
- Configure one clearly understood registered AF behavior before adding more complexity.
- Test exposure compensation with Manual exposure and Auto ISO before field use.
- Recheck assignments after a camera reset or firmware-related settings reset.

## Common Mistakes

- Removing autofocus from the shutter button before assigning and testing **AF-ON**.
- Treating a recommended assignment as confirmation of the camera's current state.
- Forgetting which AF behavior was registered.
- Expecting exposure compensation to change a fully manual exposure with fixed ISO.
- Duplicating the full control explanation inside every subject profile.
